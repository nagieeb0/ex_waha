    ExWaha.Session.Server (ExWaha v0.1.0)

    Copy Markdown View Source

    Per-account state machine.

    Holds:

    • the account struct

    • status (:stopped | :starting | :scan_qr | :working | :failed)

    • a token-bucket rate limiter (sized by warmup age)
    • jitter config

    gated_call/2 is the public API used by ExWaha.SafeSend. It serializes the rate-limit check (so two concurrent senders can't both consume the last token) but releases the GenServer before performing the HTTP call so the process is not the bottleneck.
