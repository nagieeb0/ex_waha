    ExWaha.Provider behaviour (ExWaha v0.1.0)

    Copy Markdown View Source

    Behaviour every WhatsApp provider implements.

    The facade (ExWaha) dispatches by inspecting the account struct's module's provider/0 callback (set via use ExWaha.Provider, ...) — but in practice we keep it explicit by pattern-matching on the account struct in dispatch/2.
