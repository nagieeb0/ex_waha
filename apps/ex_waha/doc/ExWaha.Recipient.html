    ExWaha.Recipient (ExWaha v0.1.0)

    Copy Markdown View Source

    Normalized recipient handle.

    Different providers expect different chat-id formats. This struct captures the canonical phone number plus an optional WAHA-style chat id, and provides helpers to render the right shape per provider.

    WAHA expects <digits>@c.us for users and <id>@g.us for groups. Meta Cloud API expects E.164 digits without +.
