# AshWaha v0.1.0 - API Reference

## Modules

- [AshWaha](AshWaha.md): Ash extension that adds a `whatsapp` DSL block to resources, plus changes
that wrap `ExWaha` calls so they participate in Ash actions, policies, and
audit.
- [AshWaha.Changes.SendText](AshWaha.Changes.SendText.md): Ash change that sends a WhatsApp text message as part of an action.
- [AshWaha.Dsl](AshWaha.Dsl.md): Spark DSL definition for the `whatsapp` block on Ash resources.
