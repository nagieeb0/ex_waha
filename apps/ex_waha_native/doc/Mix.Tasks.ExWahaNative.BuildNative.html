    Build the Go bridge binary.

    Compiles native/bridge into priv/native/whatsmeow_bridge. Requires the Go toolchain (go ≥ 1.21) on PATH.

    mix ex_waha_native.build_native

    Options

    • --target TRIPLE — cross-compile (e.g. linux/amd64, darwin/arm64). Defaults to the host platform.

    Output

    The compiled binary is placed at priv/native/whatsmeow_bridge. This path is what ExWahaNative.Bridge looks up at runtime via Application.app_dir/2.
