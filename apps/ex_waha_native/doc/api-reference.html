    Modules

    ExWahaNative

    Native Elixir provider for ExWaha — no WAHA Docker container required.

    ExWahaNative.Account

    An account on the native (whatsmeow) provider.

    ExWahaNative.Bridge

    GenServer that owns the Go bridge Port and routes traffic between Elixir callers and the whatsmeow process.

    ExWahaNative.Provider

    ExWaha.Provider implementation that routes through the Go bridge.

    ExWahaNative.Session

    Per-WhatsApp-account state machine.

    ExWahaNative.Session.Registry

    Process registry for ExWahaNative.Session instances, keyed by session name.

    ExWahaNative.Session.Supervisor

    DynamicSupervisor for per-account ExWahaNative.Session processes.

    ExWahaNative.Wire

    Length-prefixed JSON framing for the Bridge ↔ Go process protocol.
