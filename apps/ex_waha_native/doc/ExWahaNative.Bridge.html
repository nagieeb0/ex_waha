    ExWahaNative.Bridge (ExWahaNative v0.2.0)

    Copy Markdown View Source

    GenServer that owns the Go bridge Port and routes traffic between Elixir callers and the whatsmeow process.

    • Outbound: serializes %{op, args} requests to length-prefixed JSON, stores ref -> from for the reply.
    • Inbound: stream-decodes frames; routes response to the awaiting caller, dispatches event to the session GenServer.

    Crash semantics

    If the Port closes (Go process died), this GenServer exits. Its supervisor restarts it, which re-spawns the Go binary. whatsmeow re-connects every session from sqlstore. In-flight requests get {:error, :bridge_restart}.
